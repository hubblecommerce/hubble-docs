# hubble Layout

This is the default hubble layout component. It's a full responsive, user friendly interface which imports a set of
components. It's used on all pages except the checkout pages. 

## Imports
- ScrollToTop 
- TheLogo  
- TheSearchDirect 
- TheMiniCart 
- CustomerMenu 
- TheMobileMenu 
- TheMegaMenu 
- BackgroundBlur 
- TheWishlist 
- CookieNotice

## Vuex
- modApiResources
- modCookie
